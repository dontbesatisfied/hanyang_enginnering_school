#### Layout

레이아웃의 어원은 _lay something out_ 으로 펼처놓다라는 의미입니다.

때문에 레이아웃은 위젯을 포함하여 화면 요소들을 어떻게 배치할지를 결정하는 역할을 합니다.
쉽게말해서 **컨테이너** 라고 생각하시면 됩니다.
